Projects of Andrew Hyatt

Zombie Party

A top-down arcade zombie shooter written with the HTML5/JavaScript Phaser game library. It features 10 levels of increasing difficulty, sounds, reloading, ammunition, health, lava, and water. It can be played in any modern browser.

Radio Operator

A game also written with Phaser that simulates the experience of a radio operator in the early 20th century. It features a real transcript from the sinking of the RMS Titanic in morse code that the user must transcode to the latin script. Also included is a fictional transcript from Pearl Harbor.

UNIX Permission Visualizer

A simple Swing GUI written in Java that makes creating and understanding UNIX permission octals easier. It supports the special octal (setuid, etc) in addition to the three permission octals.
